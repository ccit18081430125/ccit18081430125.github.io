☀hexo搭建(一)

By Cao Yinyin

发表于 2020-04-19

一、写博客首先要先下载Node.js和Git ❤

安装node.js❤

#### 1.点击安装node.js下载，根据自身的电脑来选择是windows还是苹果或是其他。在这过程中，不要迟疑，不要怕，下不成功，从头再来！安装完成后在cmd窗口中输入node -v指令，若在Git Base Here里输入前面会$符号。安装成功将会如下显示:

继续执行指令:npm -v,若显示版本号，则安装成功。

安装Git❤

#### 若出现Git的各种用法，则表示安装成功。

二、安装Hexo ❤

#### 1.执行npm install hexo-cli -g;若出现版本号则安装成功，如图所示：

可以输入heox -v来进行检验

#### 2.创建hexo博客根目录:执行md blog
#### 3.初始化hexo博客:cd blog hexo init 若出现“Start blogging with Hexo!”字符串，则表示初始化成功。

三、博客预览❤

#### 1.执行hexo s,启动Hexo服务,出现下图则表示成功
![](/images/12.jpg)
#### 2.在浏览器输入http://localhost:4000/，看到Hexo初始页面即可。

万里长征的第一步好了！开心！
