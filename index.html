By Cao Yinyin

一、添加主题

❤ 我主题使用的是jacman主题,可参考链接，在下载之前可优先升级一下Typora,因有的主题有版本要求，下载完成后，将 blog文件夹下的_config.yml文件thems后面的landscape改为jacman。就可在cmd里依次运行hexo clean、hexo g、hexo s,可以先本地预览一下，有没有主题效果显示。

二、添加图片

❤ 我们通常会添加一些装饰让博文锦上添花，图片更是少不了。如何添加图片?不使用其他软件？写博文我用的是Typora,通过更改一些设置，使图片上传到我们的博文，为我们的博文添一份光彩。1.打开Typora 2.点击文件 3.点偏好设置 4.如图:

通过如上图所示，就能让图片上传到我们的博文，以达到美化博文的效果。

三、添加评论

❤ 我使用的jacman主题默认的多说评论，几年前已经关闭，一开始所找的都是关于多说，怎么都找不开网页，因这个评论烦恼了好多天，另辟新路，找到了畅言，关于畅言的注册，各方面都蛮顺利，一开始怕的不行，深怕他又哪时出现 了问题。这边是畅言的链接:畅言具体操作如下:

1.注册畅 言，按照它的指示，一步一步来就可以了。进入畅言官网http://changyan.kuaizhan.com/，点击右上角"免费注册"，并填写注册信息。

2.注册完后，回到畅言官网http://changyan.kuaizhan.com

首先要对邮箱里的域名白名单进行设置，如下图所示:

3.在后台-设置-通用设置-高级设置中，获取你的appid和appkey

4.在blog-themes-jacman- _config.yml文件填入对应的appid 和appkey

四、注册域名

在前面都没有什么问题了，我们就可以来注册域名了，域名我用的是阿里云的，一年只需9元 。步骤为:注册-登录-控制台-域名注册-实名认证-解析。

在这里要注意，我们在认证的过程中会出现认证通不了的情况，要注意看以下的设置有没有设置成功

1.信息模板有没有填写

2.在cmd里输入usename.github.io

五、购买云服务器

云服务器我选择的是阿里云，因为它备案只要一次，就果断选择了这个(^__^) 嘻嘻……，如果购买的话，就不要我多介绍了，因为毕竟是花钱的，肯定是会的。购买后肯定就是备案了，备案是一个很漫长的等待，如果有什么问题阿里云的后台技术人员会通过打电话的形势来跟你沟通，让你做相应的改正。接下来还是漫长的等待。注意哦:在备案过程中，需要把域名解析中断呦！备案成功后，就要在服务器上建站，具体的可以访问这个网站。进入宝塔面板后，点击文件，将public 中的文件都一个个都上传上去:可以看下图)) 根目录一定要注意路径呦，可千万不能出错，这也可能会导致后面的网页会打不开。( ⊙ o ⊙ )！

☀hexo搭建(一)

By Cao Yinyin

发表于 2020-04-19

一、写博客首先要先下载Node.js和Git ❤

安装node.js❤

#### 1.点击安装node.js下载，根据自身的电脑来选择是windows还是苹果或是其他。在这过程中，不要迟疑，不要怕，下不成功，从头再来！安装完成后在cmd窗口中输入node -v指令，若在Git Base Here里输入前面会$符号。安装成功将会如下显示:

继续执行指令:npm -v,若显示版本号，则安装成功。

安装Git❤

#### 若出现Git的各种用法，则表示安装成功。

二、安装Hexo ❤

#### 1.执行npm install hexo-cli -g;若出现版本号则安装成功，如图所示：

可以输入heox -v来进行检验

#### 2.创建hexo博客根目录:执行md blog
#### 3.初始化hexo博客:cd blog hexo init 若出现“Start blogging with Hexo!”字符串，则表示初始化成功。

三、博客预览❤

#### 1.执行hexo s,启动Hexo服务,出现下图则表示成功
![](/images/12.jpg)
#### 2.在浏览器输入http://localhost:4000/，看到Hexo初始页面即可。

万里长征的第一步好了！开心！
