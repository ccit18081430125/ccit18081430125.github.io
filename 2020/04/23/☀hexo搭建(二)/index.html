  • 2.新建一篇博文:❤
    执行以下语句:hexo new “filename” “filename”可以是自己喜欢的文件名。

    二、查看新博文

    • 清除旧数据(public 文件夹下) hexo clean❤
    • 生成新数据(public文件夹下) hexo g❤
      若结果连续出现Generated字符串，则表示成功
    • 启动hexo服务 hexo s❤

      在浏览器中输入网址，若能看见页面则表示成功！

    三、hexo博客乔迁Github

  • Github注册❤
    点击Github，选择右上角的Sign up注册

  • 在Github下新建仓库❤

    仓库名必须写为username.github.io;username为你的用户名

  • 配置ssh密钥❤
    在blog文件夹下打开Git Base Here输入ssh-keygen -t rsa -C “youremail@youremail.com“
